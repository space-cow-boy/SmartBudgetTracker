# SmartBudgetTracker

A cloud-based personal expense tracker using Firebase Authentication, Firestore, and Chart.js. Track your spending, visualize your budget, and take control of your finances.

## 🎯 Features

- **Secure Authentication** – Firebase Authentication for safe login and registration
- **Real-Time Expense Tracking** – Add, edit, and delete expenses instantly
- **Visual Analytics** – Interactive Chart.js visualizations to analyze spending patterns
- **Cloud Storage** – All your data securely stored in Firestore
- **Budget Management** – Set categories and monitor your spending limits
- **Responsive Design** – Works seamlessly on desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend**: JavaScript, HTML, CSS
- **Backend**: Firebase (Authentication, Firestore, Hosting)
- **Visualization**: Chart.js
- **Database**: Firestore (NoSQL)

## 📋 Prerequisites

Before getting started, make sure you have:

- [Node.js](https://nodejs.org/) (v14 or higher)
- npm or yarn package manager
- A [Firebase account](https://firebase.google.com/) (free tier available)
- A modern web browser
